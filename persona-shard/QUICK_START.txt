Persona Shard - Quick Start Guide
=================================

This guide will help you get started with Persona Shard development.

PREREQUISITES:
- Node.js 18+
- npm or yarn

LOCAL DEVELOPMENT:
1. Install dependencies: npm install
2. Start local network: npm run local
3. Deploy contracts: npm run deploy:local
4. Start frontend: cd frontend && npm run dev

TESTNET DEPLOYMENT:
1. Configure .env with your wallet private key and RPC URLs
2. Deploy to Sepolia: npm run deploy:sepolia
3. Update frontend config with new contract addresses

ENVIRONMENT VARIABLES:
Create a .env file in the root directory:
PRIVATE_KEY=your_private_key_here
INFURA_API_KEY=your_infura_key_here

For frontend, create frontend/.env:
VITE_WALLETCONNECT_PROJECT_ID=your_walletconnect_project_id

TROUBLESHOOTING:
- Make sure Hardhat node is running on port 8545
- Check that your wallet is connected to the correct network
- Ensure FHEVM SDK is properly loaded (check browser console)

For detailed documentation, see README.md
